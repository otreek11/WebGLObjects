# WebGLObjects
Um programa feito em WebGL para manipular objetos 3D, incluindo sua textura, iluminação e visualização. Projeto realizado como terceira avaliação para a disciplina de Computação Gráfica sob a supervisão do professor Darlan Bruno Pontes Quintanilha

# Controles
Os controles de Projeção/Animação estão presentes no HTML

- Para movimentar a câmera: WASD
- Para olhar em volta: IJKL
